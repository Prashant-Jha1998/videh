One-time on EC2 (after deploy), inside the HTTPS server block for videh.co.in add:

    root /var/www/videh-site;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }

Keep the existing location /api/ { ... } block as-is.

IMPORTANT: use `location ^~ /api/` (caret-tilde), not plain `location /api/`.
Otherwise nginx regex static rules for `*.jpg`/`*.png` steal story/chat media URLs
and return HTML 404 instead of proxying to Node.


Then run:
  sudo nginx -t
  sudo systemctl reload nginx
